# LightWeb 2.0 

古い Android デバイスでも **快適にブラウジング**できる、軽量・高速な Web ブラウザです。  
**Android 5.0（Lollipop）以上**に対応しています。

本プロジェクトは **オープンソース** で、誰でも自由にソースコードを確認・改変できます。

## ✨ 特徴

- 古い Android 端末でも動作が軽い  
- Android 5.0 以上対応  
- シンプルで無駄のない UI  
- 低メモリ・低負荷設計  
- オープンソース（自由に改変可能）

---

## 🔧 動作環境

- **対応 OS**: Android 5.0 以上  
- **対象デバイス**: 古いスマートフォン / タブレット含む  

---

## 📂 オープンソースについて

このプロジェクトはオープンソースです。

- 個人・学習・改造用途に自由に利用できます  
- Pull Request や Issue も歓迎します
